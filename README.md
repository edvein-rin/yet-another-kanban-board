# Yet Another Kanban Board

## Setup

```
yarn install
cd client
yarn install
```

## Development

### Client:

```
cd client
yarn start
```

Client is available at `https://localhost:3000`.

### Server:

```
yarn build
yarn start
```

Server API is available at port `5000`.
Server hosts client build (`client/build`) at`https://localhost:5000`.

## Deploy

Done automatically from the `main` branch.
